# Food Menu PHP (Demo)

## Langkah setup
1. Buat database MySQL bernama `foodmenu` (atau apa pun lalu ubah `config.php`).
2. Import file `db.sql` ke database tsb.
3. Copy folder proyek ini ke htdocs (XAMPP) atau public root server Anda.
4. Edit `config.php` untuk username/password DB.
5. Buka: `/public/index.php` untuk halaman pengguna.
6. Buka: `/admin/login.php` untuk dashboard (user: `admin`, pass: `admin123`).

> Ganti kredensial admin dan gunakan password hashing ketika produksi.
